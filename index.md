To set up the database with necessary tables:

1. On the `phpMyAdmin` interface, select your newly created database.

3. Click the "Choose File" button and select the SQL file from the extracted folder (usually named `database.sql`).
4. Click "Go" to import.

3. Enter your database name, username, and password (default is usually `root` with no password).
